## ‚ú® Features
- **Responsive Design:** Fully fluid layout that adapts perfectly from mobile phones to desktops using Tailwind's grid system.
- **Glassmorphism UI:** Modern "frosted glass" effects on the navbar and project overlays (`backdrop-blur`).
- **Typewriter Effect:** Custom JavaScript animation in the hero section.
- **Interactive Tech Stack:** Hover effects on skill badges using real brand colors.
- **Project Details on Hover:** "Peek" functionality using the `peer` class to reveal project specs without cluttering the UI.
- **Smooth Scrolling:** Native smooth navigation between sections.

## üöÄ Deployment
This site is hosted on **GitHub Pages**.  
To deploy your own version:
1. Fork this repository.
2. Push your changes to the `main` branch.
3. Go to **Settings > Pages** and select `main` as the source.
4. Your site will be live at `https://<your-username>.github.io/Portfolio/`.
